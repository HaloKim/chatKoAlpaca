# chatKoAlpaca
KoAlapaca + Chat

훌륭한 LLM KoAlpaca에 chat 튜닝을 할 수 있습니다.

제가 작업한 환경은 A100 80G 이며 대부분의 LLM에서 CUDA OOM이 발생합니다.

따라서 BASE LLM은 EleutherAI/polyglot-ko-1.3b 이 모델 입니다!
